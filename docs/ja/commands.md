# コマンド一覧

すべてのコマンドはプレフィックス形式（`!コマンド`）とスラッシュコマンド形式（`/コマンド`）の両方が使えます。

## 一般コマンド

告知チャンネル以外でも使用できます。

| コマンド | 説明 |
|---------|------|
| `!ping` または `/ping` | Botがオンラインかどうかを確認します。現在のバージョンを返します。 |
| `!version` または `/version` | 現在のBotバージョンを表示します。 |

## 管理者コマンド

**管理者ロールが必要です。** 告知チャンネル内でのみ使用できます。

| コマンド | 説明 |
|---------|------|
| `!list` または `/list` | 現在スケジュールされているすべての告知をジョブIDと予定日時とともに一覧表示します。 |
| `!cancel <ジョブID>` または `/cancel` | 指定したジョブIDの告知をキャンセルします。 |
| `!help` または `/help` | 管理者コマンドの一覧を表示します。 |

## コマンドの補足

### `!list` / `/list`

スケジュール済みの告知がある場合、以下の情報がEmbedで表示されます：

- ジョブID
- 投稿予定日時
- タイトル
- 内容（長い場合は省略）

告知がない場合は「予約されている告知はありません。」と表示されます。

### `!cancel <ジョブID>` / `/cancel`

ジョブIDは予約完了Embedの「ジョブID」フィールドに記載されています。

**例：**
```
!cancel abc123
```

キャンセルが成功すると「ジョブID abc123 の告知をキャンセルしました。」と表示されます。
