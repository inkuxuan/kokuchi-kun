# はじめに

VSPC BotはVRChatグループへの告知を自動スケジュールするDiscord Botです。Discordのチャンネルにメッセージを送るだけで、指定した日時にVRChatグループへ自動投稿されます。

## このBotでできること

- 告知を事前に申請し、指定日時にVRChatグループへ自動投稿
- AIによる日時・内容の自動解析（日本語テキストをそのまま送付可能）
- 管理者による承認ワークフロー
- VRChatグループカレンダーへのイベント登録

## 申請の手順

### 1. 告知チャンネルへ移動する

Botが監視している専用の告知チャンネルへ移動してください。チャンネル名や場所はサーバー管理者に確認してください。

### 2. Botをメンションして告知内容を送る

Botのメンション（`@VSPC Bot` など）を含めて、告知内容を自然な文章で送信します。

**例：**
```
@VSPC Bot
【イベント告知】
5月10日 20:00から「初心者歓迎！ VRChatお茶会」を開催します。
告知は5月8日 12:00に投稿してください。
場所: ワールド「Cozy Cafe」
```

特別なフォーマットは不要です。日時・タイトル・内容が読み取れる文章であれば大丈夫です。

### 3. Botの反応を確認する

Botがメッセージを確認すると、👀（目のリアクション）を付けて「告知リクエストを確認しました。管理者の承認を待っています。」と返答します。

### 4. 管理者の承認を待つ

管理者がリクエストを確認し、👍リアクションで承認します。承認されると、予約完了のEmbedメッセージが表示されます。

### 5. 予約完了

予約完了メッセージには以下の情報が含まれます：

- **投稿日時** — VRChatへ投稿される日時
- **タイトル** — AIが抽出したタイトル
- **内容** — AIが抽出した告知文
- **ジョブID** — キャンセル時に使用するID
