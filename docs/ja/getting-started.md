# はじめに

Kokuchi KunはVRChatグループへの告知を自動スケジュールするDiscord Botです。Discordのチャンネルにメッセージを送るだけで、指定した日時にVRChatグループへ自動投稿されます。

## このBotでできること

- 告知を事前に申請し、指定日時にVRChatグループへ自動投稿
- AIによる日時・内容の自動解析（日本語テキストをそのまま送付可能）
- 管理者による承認ワークフロー
- VRChatグループカレンダーへのイベント登録

## 申請の手順

### 1. 告知チャンネルへ移動する

Botが監視している専用の告知チャンネルへ移動してください。チャンネル名や場所はサーバー管理者に確認してください。

### 2. Botをメンションして告知内容を送る

Botのメンション（`@kokuchi-kun` など）を含めて、告知内容を自然な文章で送信します。

**例：**
```
@kokuchi-kun
告知日付：2025年1月1日 18:00

告知タイトル：「Let's Play」開催のお知らせ

告知内容：

2025年1月1日 20:00 から 21:00 まで、「Let's Play」を開催します！
みんなで一緒に遊びましょう！
途中参加、退室自由です。
デスクトップや、見学のみのご参加も歓迎です。
参加方法：・・・・・・
参加条件・人数制限：・・・・・・
```

基本下記を含めばOKです
- 告知タイトル（ポストで使います）
- 告知の発火時刻（JST）（ポストで使います）
- イベントの開始時刻（JST）（カレンダーで使います）
- イベントの終了時刻（JST）（カレンダーで使います）
- イベント名（カレンダーで使います）

### 3. Botの反応を確認する

Botがメッセージを確認すると、👀（目のリアクション）を付けて「告知リクエストを確認しました。管理者の承認を待っています。」と返答します。

### 4. 管理者の承認を待つ

管理者がリクエストを確認し、👍リアクションで承認します。承認されると、予約完了のEmbedメッセージが表示されます。

### 5. 予約完了

予約完了メッセージには以下の情報が含まれます：

- **投稿日時** — VRChatへ投稿される日時
- **タイトル** — AIが抽出したタイトル
- **内容** — AIが抽出した告知文
- **ジョブID** — キャンセル時に使用するID
